# 系统集成
## 概念
    通过结构化的综合布线系统和计算机网络技术，将各个分离的设备，功能和信息等集成到相互关联的，统一和协调的系统之中，是资源达到充分共享，实现集中、高效、便利的管理。
## 定义
    根据一个复杂的信息系统或子系统的要求，把多种产品和技术验明，并连入一个完整的解决方案的过程。
## 本质
    最优化的综合统筹设计
## 特点
- 根本目的在于满足用户需求
- 选择最合适的产品而不是最好的产品
- 技术是核心，管理和商务活动是系统集成项目成功实施的可靠保障
## 发展方向
- 产品技术服务型
- 系统咨询型
- 应用产品开发型
## 分类
- 设备系统集成
   - 智能建筑系统集成（Intelligent Building System Integration）
   - 计算机网络系统集成（Compute Network System Integration）
   - 安防系统集成（Security System Integration
- 应用系统集成
## 内容
- 客户行业知识
- 技术解决方案
- 产品技术
- 管理
- 服务

# 智能建筑系统集成
## 核心要素
- 智能建筑 BA
- 办公自动化 OA
- 通信自动化 CA
## 功能层次
- 设备级集成
- 系统级集成
- 经营管理级集成
## 完整组成
- 广播电视机卫星通信系统
- 程控电话交换系统
- 办公自动化系统
- 多媒体远程会议系统
- 综合布线
    - 智能电梯系统
    - 智能照明控制系统
    - 智能排水自控系统
    - 智能中央空调系统
    - 智能空调冷热水机组系统
    - 智能变配电系统
- 智能卡管理系统
- 保安监控报警
- 消防自动化系统
- 背景音响系统
## 组成
- 建筑设备自动化系统
    - 将建筑物或建筑群内的*电力*、*照明*、*空调*、*排水*、*防灾*、*安保*、*车库管理*等设备控制系统，以集中监视、控制和管理为目的，构成综合系统
- 通信网络系统
    - 包括*数字程控交换机*、*无线通信系统*、*卫星通信系统*、*有线广播系统*、*电视会议系统*，是建筑物内语音、数据、图像传输的基础设备，与内外部相连
- 信息网络系统
    - 由*计算机网络*、*数据库*、*服务器*、*工作站*、*网关*、*路由器*等网络设备及软件构成
- 综合布线系统
    - 建筑物或建筑群内部之间的传输网络
## 存在的问题
- 子系统相互孤立
- 重点不突出
- 理解错位

# 安防系统集成
## 概念
    利用红外线、音视频、探测、控制、通信等多种科学技术，采用各种安防产品和设备，给人们提供一个安全的生活和工作环境的系统、达到实现预警、事后控制和处理的效果，保护建筑内外的人身以及生命财产安全
## 狭义组成
- 前端报警探头
- 传输部分
- 报警主机
## 广义组成
- 闭路电视监控系统（CCTV，Close Circuit Television）
    - 组成
        - 摄像
        - 传输
        - 控制和显示
- 门禁系统（Access Control System）
    - 类型
        - 密码门禁系统
        - 刷卡门禁系统
        - 生物识别门禁系统
    - 组成
        - 独立门禁系统
            - 可存储记录数据门禁
            - 不可存储记录数据门禁
        - 联网门禁系统
- 防盗报警系统（Intruder Alarm System）
    - 探测器
        - 漏报率
        - 探测率
        - 误报率
        - 探测范围
        - 传输方式和传输距离
        - 探测灵敏度
        - 功耗
        - 工作电压和电流
        - 工作环境
        - 工作时间
    - 现场控制器
        - 接受带地址的报警信息
        - 通过编程确定防区性质
        - 通信功能
    - 监控中心
        - 接收现场报警、显示打印报警信息，显示报警现场平面图形等等
## 设计原则
- 自动防故障
- 可扩充
- 不引人注意且易于维修
- 遵循标准
- 无盲区

# 计算机网络系统集成
## 概念
	根据应用的需要，将硬件设备、网络基础设施、网络设备、网络系统软件、网络基础服务系统、应用软件等组织为一体，使之成为能够满足设备目标并且具有优良性能价格比的计算机网络系统的过程
## 步骤
- 网络系统规划和需求分析
- 逻辑网络设计
- 物理网络设计
- 设备订购和安装
- 服务器安装和配置
- 网络系统测试
- 网络安全和网络管理
- 网络系统验收
    - 后期维护
- 设计原则
    - 开放性和标准化原则
    - 实用性和先进性兼顾原则
    - 无瓶颈
    - 可用性原则
    - 安全第一原则
